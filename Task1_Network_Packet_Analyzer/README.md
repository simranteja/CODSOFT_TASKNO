
# Task 1 - Network Packet Analyzer

## Objective
Develop a Python application to capture and analyze network packets in real-time.

## Description
This project captures live network traffic and displays useful information about each packet such as:
- Source IP Address
- Destination IP Address
- Protocol (TCP / UDP / ICMP)
- Packet Length
- Packet Summary

It helps in understanding how data travels over a network.

## Tools Used
- Python
- Scapy
- Npcap (for Windows packet capture support)

## Features
- Live packet capturing
- IPv4 and IPv6 support
- Protocol detection
- Packet analysis in real-time

## How to Run
python packet_analyzer.py

## Output
Displays real-time captured network packets with details in terminal.
